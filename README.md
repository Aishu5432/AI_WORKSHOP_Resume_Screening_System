# AI Resume Screening System

## Overview

This project uses LangChain, ChromaDB, Gemini and RAG to:

- Parse resumes
- Store resume embeddings
- Match resumes with job descriptions
- Rank candidates
- Answer recruiter questions

## Technologies

- Python
- Streamlit
- LangChain
- ChromaDB
- Gemini
- RAG

## Setup

1. Create virtual environment
2. Install requirements
3. Add API key to .env
4. Place resumes inside resumes folder
5. Run:

streamlit run app.py
